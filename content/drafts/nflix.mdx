This is a MERN stack application. Essentially a fullstack application using
[MongoDB](https://www.mongodb.com/) and ExpressJS in the backend and
[React](https://reactjs.org/) in the frontend.

You can visit the live application [here](https://nflix-app.herokuapp.com/)
(you may have to wait for a couple of seconds before the server becomes
active).

## Tech stack

- ReactJS
- Bootstrap
- MongoDB
- ExpressJS
- Heroku

## Features

- Faceted search using MongoDB aggregation pipeline
- Pagination using iterations over MongoDB cursor return from the query

## Challenges
